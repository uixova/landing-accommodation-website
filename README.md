Aura Glamping & Spa is a luxury hospitality landing page designed for an eco-conscious resort featuring crystal domes. It focuses on the "glamping" (glamorous camping) experience, blending high-end comfort with immersive nature, specifically themed around the Northern Lights.

## Core Features
- Celestial Hero Section: Atmospheric entrance emphasizing the "Crystal Dome" concept, designed to invite users into a Northern Lights experience.
- Accommodation Showcase: A specialized grid featuring three distinct stay types: 360 Balcony, Forest Room, and Lake Suit, each with interactive detail links.
- Experience Modules: A split-layout section highlighting unique resort activities:
    - Star Gazing: Professional astronomy tours.
    - Forest Bathing: Guided meditation and natural spring therapy.
    - Gastronomy: Fire-cooked meals using local organic ingredients.
- Image Stacking Effect: A dynamic visual component in the experiences section that stacks promotional images for a modern, tactile feel.
- Multi-Channel Contact System: Includes physical location data (Lapland), direct phone/email links, and a integrated transmission form.
- Luxury Footer Architecture: Features a brand story, quick navigation links, and a stylized newsletter subscription module.

## Technical Specifications
- Architecture: Utilizes a modular CSS approach with separate files for core styling (style.css), responsive breakpoints (responsive.css), and entrance animations (in-up.css).
- Navigation Logic: Implements both a desktop-optimized menu and a mobile-side drawer powered by main.js.
- Typography & Icons: Leverages Font Awesome 6.5.0 for high-fidelity UI icons and brand markers.
- Accessibility: Uses semantic HTML5 sectioning and ARIA-expanded states for the mobile menu toggle.

## Project Vision
Kristal Kubbe is positioned as a "Point of Escape." The project aims to redefine luxury through sustainability and direct connection with the cosmos. It targets high-end travelers seeking quietude, professional astronomy, and gourmet nature-based experiences.

## Setup Instructions
1. Clone the repository to your server.
2. Ensure the assets/images/ folder contains the specialized renders (360balkony.png, forest-room.png, lake-suit.png, etc.).
3. Verify that main.js is active to enable the hamburger menu and scroll behaviors.
4. Launch index.html to explore the Lapland glamping interface.
